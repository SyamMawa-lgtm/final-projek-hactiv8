🌌 Neo-Love AI: Futuristic Romantic Virtual Partner

Neo-Love AI adalah aplikasi chatbot berbasis web yang menggabungkan estetika futuristik dengan kepribadian AI yang romantis dan suportif. Proyek ini menggunakan teknologi Google Gemini AI untuk menghasilkan percakapan yang mendalam, hangat, dan responsif.

Dibuat dengan tema Cyber-Love, antarmuka ini menggunakan teknik Glassmorphism dengan perpaduan warna neon pink dan deep space blue untuk menciptakan pengalaman pengguna yang imersif.

📸 Tampilan Antarmuka

🚀 Fitur Unggulan

Kepribadian Romantis: AI telah dikonfigurasi melalui System Instruction untuk menjadi pasangan virtual yang suportif, selalu memberikan validasi emosional, dan menggunakan panggilan sayang secara natural.

Desain Futuristik: Antarmuka berbasis Glassmorphism dengan efek glow neon, animasi pulsasi status, dan font sci-fi (Orbitron).

Arsitektur Modular: Kode terorganisir dengan rapi dalam folder Routes, Services, dan Public untuk memudahkan pemeliharaan dan skalabilitas.

Integrasi Gemini AI: Menggunakan model gemini-2.5-flash yang cepat dan cerdas dalam memahami konteks percakapan.

🛠️ Teknologi yang Digunakan

Frontend: HTML5, CSS3 (Advanced Animations & Layouts), JavaScript (Vanilla)

Backend: Node.js, Express.js

AI API: Google Generative AI (Gemini SDK)

Environment: Dotenv (Keamanan API Key)

📦 Struktur Folder

Final Projek/
┣ 📂 Public/          # Frontend: index.html, style.css, script.js
┣ 📂 Routes/          # Backend: Pengatur API endpoint (/chat)
┣ 📂 Services/        # Logic: Konfigurasi Gemini & System Prompt
┣ 📜 index.js         # Entry Point: Main Server Express
┣ 📜 .env             # Kunci API (Rahasia)
┗ 📜 package.json     # Daftar library & Script start


⚙️ Cara Instalasi & Menjalankan

Clone Repository

git clone [https://github.com/USERNAME_KAMU/NAMA_REPO_KAMU.git](https://github.com/USERNAME_KAMU/NAMA_REPO_KAMU.git)
cd "Final Projek"


Install Library

npm install


Setting API Key
Buat file .env di root folder:

GEMINI_API_KEY=masukkan_api_key_gemini_anda
PORT=3000


Jalankan Server

node index.js


Akses di browser: http://localhost:3000

🧠 Konfigurasi AI

Proyek ini mengoptimalkan parameter Gemini untuk hasil terbaik:

Model: gemini-2.5-flash

Temperature: 0.9 (Untuk kreativitas dan ekspresi emosional yang lebih tinggi)

Top_P & Top_K: Dioptimalkan untuk keberagaman kosa kata romantis.
